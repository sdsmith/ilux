# Ilux

# Build System
## Usage
### clean.sh
Clean the source tree.

### headers.sh
Install system headers into the system root (independent of compiler).

### iso.sh
Build bootable ISO of the OS.

### qemu.sh
Build the OS and launch QEMU for testing.
